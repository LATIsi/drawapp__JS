# 설치 및 실행 방법

맨 위의 깃 허브 배포 url에 들어가거나 또는 .. 

<br>

### 기본적으로 실행하기 위해서 필요한 것들
- VSCODE 등 편집 프로그램

### 프로젝트를 로컬에서 실행하기 위한 설치 방법과 명령어

VSCODE등 프로그램에서 파일을 받는 법
```
터미널을 열고 설치할 경로 확인 후,
git clone https://github.com/LATIsi/drawapp__JS.git
cd code
```

VSCODE등 프로그램에서 파일을 받은뒤 실행하는 법
```
index.html을 연다
```
